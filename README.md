# 🏡 StayNest

A full-stack Airbnb-inspired accommodation booking platform built using Node.js, Express.js, MongoDB, EJS, Tailwind CSS, and Mapbox.

## ✨ Features

- 🔐 User Authentication
- 👤 Guest & Host Roles
- 🏠 Add/Edit/Delete Property
- ❤️ Favourite Homes
- 🔍 Search by Location
- 💰 Price Filter
- 📅 Dynamic Booking System
- 💳 Fake Payment Gateway
- ⭐ Reviews & Ratings
- 👤 User Profile
- 🗺️ Mapbox Integration
- 📱 Responsive Design

- ## 🛠 Tech Stack

### Frontend
- HTML
- EJS
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### API
- Mapbox

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/naitiky27-sys/StayNest-FullStack.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
MONGODB_URI=MongoDBURI
MAPBOX_TOKEN=MapboxToken
SESSION_SECRET=MySecret
```

Run the project

```bash
npm start
```
